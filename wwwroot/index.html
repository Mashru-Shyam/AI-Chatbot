IntelliChat

IntelliChat is a state-of-the-art conversational assistant powered by LLaMA-3 for advanced Natural Language Processing (NLP). It seamlessly interacts with users and retrieves real-time information from a dynamic dataset.

Key Features

  • ✔ General Question Answering
  • ✔ Dynamic Data Handling
  • ✔ Scheduling & Viewing Appointments
  • ✔ Viewing Prescription & Payment Details
  • ✔ Insurance Information Retrieval
  • ✔ Secure OTP Login & Authentication
  • ✔ AI-Powered Intent Recognition & Entity Extraction

How It Works?

IntelliChat leverages intent recognition and entity extraction to process user queries efficiently:

  1. Step 1: The user interacts with the chatnot.
  2. Step 2: The AI model classifies the intent and extracts relevant data.
  3. Step 3: It fetches required information from the SQL database.
  4. Step 4: The chatbot provides an instant, accurate response.
  5. Step 5: Users can perform actions like setting appointments, checking prescriptions, and more.
